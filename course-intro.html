Coding Overview

  • For marketing analytics, we need a way to
    • analyze data => python
    • store and represent data => use Dataframe

    • report results => Quarto

For a demonstration of a simple line plot using plotly, see Figure 1.

import plotly.graph_objects as go

# Generate sample data
x = [1, 2, 3, 4, 5]
y = [10, 15, 13, 17, 14]

# Create a line chart
fig = go.Figure()

fig.add_trace(go.Scatter(
    x=x, y=y,
    mode='lines+markers',
    name='Line Plot',
    line=dict(color='royalblue', width=2),
    marker=dict(color='royalblue', size=8)
))

# Customize the layout
fig.update_layout(
    title='Simple Line Chart',
    xaxis_title='X Axis',
    yaxis_title='Y Axis',
    template='plotly_white'
)

# Show plot
fig.show()

from datetime import datetime

    {

            datetime(2025, 1, 1),
            datetime(2025, 1, 2),
            datetime(2025, 1, 3),
        ],
        "float": [4.0, 5.0, 6.0],
        "string": ["a", "b", "c"],
    }
)

print(df)
